## React + TypeScript + Vite

### get started with the frontend using
`cd todo-app/client`

### install npm modules
`npm i`

### create .env file and populate it with server ip/api
`mv .env.template .env`
set .env values accordingly

### build 
`npm run build`

### after build
`cd dist`

### run the vite server
`npm run preview -- --host --port 80`
set port accordingly
